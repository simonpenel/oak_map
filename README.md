# oak_map


cloner le dépot  https://github.com/simonpenel/oak_map

`cd  oak_map`

Si il s'agit de la première installation, (cf https://openlayers.org/doc/quickstart.html) :

```
npm create ol-app my-app
cd my-app
cp ../main.js .
cp  ../style.css .
cp  ../index.html .
cp -r  ../data .
npm start
```
